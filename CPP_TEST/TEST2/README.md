## Problem: money change

You are given the denominations (coins) to give a change back for a target amount:

1. Return a vector of the counts of every used coin.
2. Return the remaining amount that could not be changed with the available coins 

### Input Example:
Enter the amount you want to change: 17

### Output Example:
Problem: Coin Change Problem
 
Coins used to make 17: 
Coin: 10, Count: 1
Coin: 5, Count: 1
Coin: 2, Count: 1
Coin: 1, Count: 0
